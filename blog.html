What are the different ways to select an element in the DOM?
There are five ways to select an element in the DOM .

1. document.getElementById. Selects one element, Returns a single element,

2. document.getElementByTagName. Selects all elements with the given tag, Returns a live HTMLCollection

3. document.getElementByClassName. Selects multiple elements, Returns a live HTMLCollection, Access elements by index

4. document.querySelector Selects the first matching element , Accepts any valid CSS selector. And give the first element from selector

5. document.querySelectorAll Selects all matching elements , Returns a static NodeList , We can use forEach loop on querySelectorAll
